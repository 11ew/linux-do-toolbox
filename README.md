# Linux.do 工具箱

一个功能丰富的 [linux.do](https://linux.do) 油猴脚本，通过悬浮球提供个人信息查看、时间线浏览、快速回复、自动刷贴等功能。

## 功能概览

### 个人信息
- 等级升级条件一览（已满足/未满足直观展示）
- Credit 积分查询
- CDK 兑换码信息（通过 iframe Bridge 跨域获取）
- 信息板块支持一键跳转：未登录点击直达登录页，已登录点击直达详情页（Credit/CDK）

### 时间线
- 浏览近期帖子，已读/未读标签区分
- 无限滚动自动加载，无需手动翻页
- 帖子淡入动画
- 点击时间线帖子跳转后，自动打开小助手并定位到当前帖子位置，便于继续向下浏览

### 快速回复
- 回复模板管理（添加/删除，持久化存储）
- 可选自动点赞（给楼主）、自动书签
- 选择模板后自动执行所有勾选操作

### 自动刷贴
- 移植自「小助手」的成熟方案
- 随机选帖，偶尔上滑/暂停，模拟真实阅读
- 累计浏览 + 定时休息（15分钟浏览 / 2分钟休息）
- 跳过已读帖子
- 导航守护 + 卡死恢复

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击下方链接安装脚本：

   **[安装 Linux.do 工具箱](https://raw.githubusercontent.com/11ew/linux-do-toolbox/main/Linux.do%20%E5%B7%A5%E5%85%B7%E7%AE%B1.user.js)**

3. 访问 [linux.do](https://linux.do)，右下角会出现悬浮球

## 使用方式

- **点击悬浮球** 打开/关闭面板
- **拖拽悬浮球** 移动位置（自动吸附屏幕边缘）
- **悬停悬浮球** 显示完整球体
- **ESC 键** 显示/隐藏悬浮球
- **面板内切换 Tab** 查看不同功能
- **点击主题按钮** 循环切换 11 种主题
